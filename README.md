# GLBI Simulator

**Guaranteed Livable Basic Income Calculator for Canada**

A comprehensive web-based simulator for calculating Guaranteed Livable Basic Income (GLBI) benefits based on the Parliamentary Budget Officer (PBO) report from February 2025.

## Live Demo

Visit: **simulator.glbi.ca**

## Quick Start

No build process required. Simply open `index.html` in a web browser or deploy to any static web host.

```bash
# Clone the repository
git clone https://github.com/jvm2020/glbi.git
cd glbi

# Open in browser
open index.html
```
